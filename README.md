# AI-Powered News Analysis Tool

## Features
- AI news summarization
- Validation of relevance & accuracy
- User authentication
- Query history
- Export summaries
- Custom themed UI

## Setup
1. Create `.env` with GROQ_API_KEY
2. Install dependencies
3. Run `save_chain_config.py`
4. Start Streamlit app
